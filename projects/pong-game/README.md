# Pong Game MVP

Neon-themed Pong with multiple game modes, particle effects, screen shake, and Web Audio synth beeps.

## Run
Just open `index.html` in a browser.

## Controls
- **Left paddle:** W / S (multiplayer)
- **Right paddle:** Arrow Up / Down
- **Single player:** You control the right paddle; AI controls the left
- **Touch:** Tap/drag left/right side of the screen to move paddles

## Modes
- Classic — first to 11
- 3 Lives — last one standing
- Giant Ball — 3x ball size
- Speed Demon — ball accelerates on hits
- Multi-Ball — 3 balls at once
- Tiny Paddles — half size paddles
